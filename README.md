# CoolRoute Dubai: Heat-Safe Mobility Analytics 🇦🇪

This repository contains the data analytics framework and interactive dashboard validating the market entry strategy for **CoolRoute Dubai**—a heat-safe smart mobility app. 

## Project Objective
To perform descriptive and diagnostic analytics on a synthetic dataset reflecting extreme summer temperatures in Dubai, validating how heat drives user adoption of shaded routes and premium cooling kits. 

## Files in this Repository
* `coolroute_raw_data.csv`: The initial synthetic dataset simulating app usage.
* `coolroute_cleaned_data.csv`: The processed dataset with handled anomalies and missing values.
* `coolroute_app.py`: The Streamlit application containing the interactive visualizations.
* `requirements.txt`: Required Python dependencies.

## How to Run Locally
1. Clone this repository.
2. Install requirements: `pip install -r requirements.txt`
3. Run the dashboard: `streamlit run coolroute_app.py`
